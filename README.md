This project is a special goods ordering application for bolts and nuts,
which is designed to make it easier for users to choose and order various types and sizes of bolts according to their needs.

This application is built using Android Studio with the Java programming language and provides an imaginative interface for a comfortable and efficient user experience.

## The main features of this application include:

* Display a list of bolt and nut products with various sizes and specifications.
* An ordering system that is easily accessible to users.
* Order summary and details of product prices and total payments.

This application is expected to help users speed up and simplify the process of ordering bolts and nuts with organized and detailed choices.

Home Screen ‚Äì Displays a list of bolt and nut categories with search and filter options to make it easier for users to find the products they need.
Product Detail Screen ‚Äì Users can view complete information about the selected product, including images, sizes, specifications, and prices per item.
Cart Screen ‚Äì Collects all items that have been selected by the user, allowing them to review and edit the order before checking out.
